# Job Skill Matcher 🔍

**Job Skill Matcher** is a web-based application designed to help **job seekers** evaluate how well their skills match a given **job description**.  
It analyzes resume text and job requirements to generate a **skill match percentage** and highlight **missing or unmatched skills**, enabling users to improve their job readiness.

📌 **GitHub Repository:** https://github.com/HarithaNetha8/job-skill-matcher

---

## 🚀 Why This Project?

Many candidates apply for jobs without knowing whether their skills truly align with job requirements.  
This project solves that problem by:

- Identifying **matching skills**
- Highlighting **skill gaps**
- Providing a **clear match score**
- Helping candidates improve resumes strategically

This project is especially useful for:
- Final-year students
- Freshers
- Resume optimization

---

## ✨ Features

- 📄 **Resume Skill Analysis**
- 📋 **Job Description Parsing**
- 📊 **Skill Match Percentage**
- ❌ **Missing Skill Identification**
- ⚡ **Fast & Responsive UI**
- 🌐 **Deployed & Accessible Online**

---

## 🧠 How It Works

1. User inputs **resume text**
2. User inputs **job description**
3. Backend processes both inputs
4. Skills are extracted and compared
5. Result shows:
   - Match percentage
   - Matched skills
   - Missing skills

## ⚙️ Installation & Setup

### Prerequisites

Ensure you have:
- Node.js (v14 or above)
- npm or yarn
- Git

---

## 🔧 Clone Repository

git clone https://github.com/HarithaNetha8/job-skill-matcher.git
cd job-skill-matcher

## ▶️ Frontend Setup
cd client  
npm install
npm start
Runs the frontend at:
http://localhost:3000

## ▶️ Backend Setup
cd server
npm install
npm run dev
Runs backend at:
http://localhost:5000

## 📡 API Endpoints

| Method | Endpoint      | Description                              |
| ------ | ------------- | ---------------------------------------- |
| POST   | `/api/match`  | Match resume skills with job description |
| GET    | `/api/health` | Server health check                      |
